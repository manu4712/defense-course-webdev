Learn the fundamentals of HTML

Introduction to HyperText Markup Language

HTML stands for HyperText Markup Language—the standard language for creating and structuring web content. It forms the backbone of every website you visit.

What is HTML?

HyperText: The ability to create links connecting web pages, forming the interconnected "web" of information we know today.

Markup Language: Uses "tags" to surround content, giving it meaning and structure. You're essentially "marking up" plain text to tell browsers how to display it.

Historical Note: The first website ever created is still accessible at https://info.cern.ch/hypertext/WWW/TheProject.html—a simple HTML page that started the World Wide Web.

HTML Document Structure

Every HTML document follows a basic skeleton:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>

Core HTML Elements

1. Headings (h1 to h6)

Headings create hierarchical structure and importance levels in your content.

Heading 1 - Main Title (Most Important)

Heading 2 - Major Section

Heading 3 - Sub-section

Heading 4 - Sub-sub-section

Heading 5 - Minor heading
Heading 6 - Least Important

Rules:

  • Use only one h1 per page for the main title
  • Follow sequential order—don't skip levels
  • Headings provide structure for search engines and screen readers

2. Paragraphs

The p tag groups related sentences into readable blocks.

This is the first paragraph containing related sentences about a topic. Browsers automatically add spacing around paragraphs.

This is a separate paragraph. Notice the automatic spacing between this and the previous paragraph.

Best Practice: Use multiple p tags instead of multiple line breaks for better semantic structure.

3. Horizontal Rules

The hr tag creates thematic breaks between content sections.

This section discusses one topic.

This section covers a completely different topic.

4. Line Breaks

The br tag forces text to continue on the next line within the same paragraph.

221B Baker Street
London, England
United Kingdom

When to Use: Addresses, poems, or anywhere you need single line breaks without paragraph spacing.

5. Lists

HTML supports both ordered (numbered) and unordered (bulleted) lists.

Unordered Lists (bullets):

  • Tea Bag
  • Water
  • Sugar
  • Milk

Ordered Lists (numbers):

  1. First, boil the water in a kettle
  2. Add tea bag, sugar and milk into it
  3. Keep boiling it for 5 minutes
  4. Serve the tea by pouring it into cup

Structure: Each list item (li) must be nested inside either ul or ol tags.

6. Links (Anchor Tags)

Links make the web "hypertext" by connecting pages together.

Link Targets:

  • target="_self": Opens in same window (default)
  • target="_blank": Opens in new window/tab

Key Terms

  • Element: Complete HTML component including opening tag, content, and closing tag
  • Tag: Keywords enclosed in angle brackets that define elements
  • Attribute: Additional information provided within opening tags
  • Nesting: Placing elements inside other elements to create hierarchy
  • Semantic HTML: Using elements based on meaning rather than appearance
  • Self-closing Tag: Elements that don't contain content (like <img> and <br>)
  • Hyperlink: Clickable connection to another resource using <a> tags
  • Alt Text: Description of images for accessibility and when images don't load
